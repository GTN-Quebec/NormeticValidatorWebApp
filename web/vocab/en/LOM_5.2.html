Vocabulary for the LOM field 5.2

Source: http://www.normetic.org/vdex/typeressourcev1_2.xml

evaluation (évaluation)
Use this term for any resource intended to provide measurements of learning either during training or at the end of a sequence of contents.
animation
This term refers to a resource which consists in an animated illustration of a concept, phenomenon or process.
tutorial (tutoriel)
Is used when the resource serves to guide users in learning and implementing application software or procedures.
glossary (glossaire)
When the resource can be used as an index of terms for which known synonyms or definitions are provided.
guide
Refers to a resource used as teaching material that presents the fundamentals of an art, science or technique.
reference material (matériel de référence)
Qualifies a resource consisting in a reference tool, source of information or data.
tools (outils)
Describes a resource that provides users with tools to complete a specific task within a broader process. For example, a spreadsheet that provides Canadian dollar equivalents for foreign currency amounts or a tool that displays universal physical constants in the units of measurement of a specific system of units.
simulation
To be used when the resource is intended to simulate certain processes or the conditions of certain situations.
lecture/presentation (lecture/présentation)
To be used for any ressource intended to present textual information, including hypertext, and oral or written communications, except when the text is used for purposes involving one or more other terms of the current vocabulary (e.g. use “evaluation” in the case of an examination questionnaire).
