Vocabulaire permis pour une langue selon ISO 639-1, ISO 639-2 et ISO 3166

LanguageID = Langcode ("-"Subcode).

Langcode est un code de langue de deux ou trois lettres, défini par ISO 639:1 et :2 et Subcode (qui peut survenir un certain nombre de fois de façon arbitraire) est un code de pays défini par ISO 3166
